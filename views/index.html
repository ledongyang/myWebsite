About

I've been programming during my four college years, and successfully graduated from SUNY University at Buffalo with B.S. degree in Computer Engineering in 2016. I'm always a fan of web technologies, and decided to self-train / self-learn to become a web developer with my programming knowledge and skills learned from school. Nowadays, web developers use variety of frameworks, databases, and technologies to build web apps. I'm specialized in MEAN stack, which stands for MongoDB, Express, Angular, and Node.js.

But I'd love to learn other technologies like Ruby on Rails, Django, Flask, MySQL, PostgreSQL, and etc. I believe it doesn't matter what technologies a developer are using right now, it is more important to learn how to use newer or more advanced technologies. That is why working with technology is so exciting but challenging at the same time. I learned MEAN from ground up, and had successfully deployed a number of express apps on Heroku. Please see the section below for more information about my most recent projects.
